*취업 관련 이것저것*
크게 2경로 
1) solution -> 제품 특화 개발 => 다양한 경험 부족으로 이어질 수 있음
2) SI(System Integration) = 시스템 통합
- SW, program, server 등 장비까지 다루며 시스템 구축 (실패시 risk가 큼)
보통 국가: 프로젝트 -> 대기업 의뢰 -> 1차 벤더회사 (직원수 70~100)
- 국가에서 한 장소를 지정해줌 (집중도 & 보안 이슈)
- 2차, 3차 벤더 회사들에게 program 개발만 의뢰하는 경우 많음

용어정리
- dll = dynamic link library : C 기반 windows에서 많이 씀
- IDE (Integrated Development Environment) : 대표적인 3개 eclipse, VScode, IntelliJ(유료)
- JDK (Java Development Kit) = 개발도구
	- JRE (Java Runtime Environment) = 실행환경									------> only 실행만 가능
		- JVM (Java Virtual Machine) = 가상 메모리 장치 
		   - 메모리의 일부를 할당함 = 마치 Java의 영역
		   - 보안에 강함, 가비지콜렉션까지 수행해줌

- class : Java의 가장 기본 단위
=> 명명법) 대문자+영문자로 시작할것 / 공백X
	*) 소문자로 시작하는 class 만들어도 버그 발생 X but 대문자로 쓰는 걸로 약속되어 있음 (권고사항)
- package : 관련있는 class들을 모아둔 폴더
- module : package들을 묶어둔 것
